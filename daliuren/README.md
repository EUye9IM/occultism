# Daliuren（大六壬）

大六壬子 crate，提供中国传统大六壬占卜的核心数据类型与算法。

## 依赖

- [shushu](../shushu/) — 数术基础库，提供天干、地支、五行等基础类型

## 设计目标

- **类型安全**：利用 Rust 类型系统对六壬概念（天地盘、四课、三传等）进行精确建模
- **可组合**：各模块独立，支持按需组合不同流派的排盘方式

## 模块规划

- [ ] 天地盘（天盘/地盘）
- [ ] 四课
- [ ] 三传（九宗门：贼克、比用、涉害、遥克、昴星、别责、八专、伏吟、反吟）
- [ ] 十二天将（贵人、腾蛇、朱雀、六合、勾陈、青龙、天空、白虎、太常、玄武、太阴、天后）
- [ ] 神煞
- [ ] 课经

## 构建

```bash
cargo build -p daliuren
cargo test -p daliuren
```
